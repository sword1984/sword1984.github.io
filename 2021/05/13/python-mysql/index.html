数据类型

规定了列可容纳何种数据类型（data_type）。下面的表格包含了SQL中最常用的数据类型：

数据类型 描述
integer(size)
int(size)
smallint(size)
tinyint(size)		 仅容纳整数。
在括号内规定数字的最大位数
decimal(size,d)
numeric(size,d)		 容纳带有小数的数字。
“size”规定数字的最大位数，”d”规定小数点右侧的最大位数。
char(size) 容纳固定长度的字符串(可容纳字母、数字以及特殊字符)。
在括号中规定字符串的最大长度
varchar(size) 容纳可变长度的字符串(可容纳字母、数字以及特殊字符)。
在括号中规定字符串的最大长度。
date(yyyymmdd) 容纳日期
